# **App Name**: Emirates Law Hub

## Core Features:

- Legislation Search: Enable users to search for UAE legislation using keywords.
- Legislation Browsing: Allow users to browse legislation by category (e.g., Regulatory Decision, Executive Regulations, Federal Law, Residence, Work).
- Legislation Details: Display detailed information for each piece of legislation, including its full text, date of enactment, and related documents.
- Statistics Dashboard: Show statistics on different types of legislation.

## Style Guidelines:

- Primary color: Deep teal (#008080) to convey authority and trust.
- Background color: Light gray (#F0F0F0) for a clean and modern look.
- Accent color: Soft gold (#D4AF37) to highlight important information and calls to action.
- Headline font: 'Playfair', a modern sans-serif with an elegant, high-end feel; body font: 'PT Sans' for longer text.
- Note: currently only Google Fonts are supported.
- Use simple, professional icons to represent different types of legislation.
- A clean, structured layout that provides clear hierarchy for different legislation sectors. Focus on readability and ease of navigation.
